Motion Segmentation Based on Subspace Clustering

 

This project was an entry for the 2015 China Post-Graduate Mathematical Contest in Modeling. It aimed to segment different moving objects in a video sequence, using subspace clustering methods.

Subspace Clustering Introduction

Clustering refers to techniques that find a low-dimensional representation of a high-dimensional data set. Conventional techniques, such as principal component analysis (PCA), assume that the data are drawn from a single low-dimensional subspace of a high-dimensional space. Such approaches have found widespread applications in many fields, e.g., pattern recognition, data compression, image processing, and bioinformatics.

In practice, however, the data points could be drawn from multiple subspaces, and the membership of the data points to the subspaces might be unknown. For instance, a video sequence could contain several moving objects, and different subspaces might be needed to describe the motion of different objects in the scene. Therefore, there is a need to simultaneously cluster the data into multiple subspaces and find a low-dimensional subspace fitting each group of points. This problem is known as subspace clustering.

Clustering of Linear Subspaces

It is a relatively simple problem to cluster the data sampled from several linear subspaces. We solved this kind of problem by applying Generalized Principal Component Analysis (GPCA) with voting mechanism. Figure 1-a shows a set of sample points in R^3 drawn from a union of three subspace: two lines and a plane. The clustering results are shown in Figure 1-b.

Motion Segmentation

Motion segmentation is the separation of objects with different motion in video, and it is an indispensable step in the understanding and reconstruction of dynamic scenes.

Feature-point trajectory based method is widely used. This method firstly uses the standard tracking method to extract feature points of different trajectories of moving objects in the video, then segments different feature points corresponding to different motion trajectory in the scene.

Considering that GPCA method is vulnerable to data noise, Sparse Subspace Clustering (SSC) was adopted to solve motion segmentaion.

SSC is based on the fact that each point in a union of subspaces has a Sparse Representation （SR） with respect to a dictionary formed by all other data points. In general, finding such a SR is NP hard. Our key contribution is to show that, under mild assumptions, the SR can be obtained ’exactly’ by using an optimization technique. The segmentation of the data is obtained by applying spectral clustering to a similarity matrix built from this SR. The result is shown in the following figure.
